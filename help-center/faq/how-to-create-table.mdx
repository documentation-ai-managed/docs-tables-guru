## Overview

Easily create tables to organize data in TABLES GURU.

## Steps

1. Click **New Table** in the sidebar
2. Enter table name and description
3. Select column types (Text, Number, Date, etc.)
4. Click **Create** to finish

## Tips

Add or modify columns later from the table settings.
