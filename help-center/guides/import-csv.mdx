---
title: Importing Data from CSV
description: How to quickly import CSV files into TABLES GURU.
---

## Overview

Import CSV data into any table for fast setup.

## Steps

<Steps>
  <Step title="Open Table">
    Select the table you want to import data into.
  </Step>
  <Step title="Click Import">
    Use the **Import** button at the top right.
  </Step>
  <Step title="Upload CSV File">
    Choose your `.csv` file from your computer.
  </Step>
  <Step title="Map Columns">
    Match CSV columns to your table fields.
  </Step>
  <Step title="Finish Import">
    Click **Import**. Your data will appear instantly!
  </Step>
</Steps>
