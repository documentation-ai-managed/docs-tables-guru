---
title: Can't See My Table
description: Fix missing tables in TABLES GURU.
---

## Symptoms

- Table not visible in dashboard
- Table recently created but missing

## Causes

- Table not saved
- Filters or views hiding tables

## Solution

<Steps>
  <Step title="Refresh the Dashboard">
    Click the Reload button in your sidebar or refresh browser.
  </Step>
  <Step title="Check Table Filters">
    Make sure no filters are hiding your table. Reset filters.
  </Step>
  <Step title="Confirm Table Creation">
    Ensure you clicked **Create** and saw the confirmation message.
  </Step>
</Steps>
